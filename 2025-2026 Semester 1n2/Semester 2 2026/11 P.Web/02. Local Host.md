## 1. Apa itu Localhost?

Sebelum membuat website yang bisa diakses semua orang di internet (online), developer biasanya membuat dan menguji website tersebut di komputer mereka sendiri. Proses ini disebut dengan pengembangan lokal (*local development*).

* **Localhost:** Istilah/nama domain yang merujuk pada komputer yang sedang kita gunakan saat ini. Secara teknis, localhost dialihkan ke alamat IP khusus (IP Loopback), yaitu `127.0.0.1`.
* **Web Server Lokal:** Agar komputer kita bisa membaca file web (seperti PHP) dan mengelola database (seperti MySQL), komputer memerlukan software khusus. Di sinilah XAMPP dan Laragon berperan.

---

## 2. Mengenal Tool: XAMPP vs Laragon

Kedua software ini gratis dan berfungsi untuk mengubah komputer kita menjadi server lokal. Namun, keduanya memiliki pendekatan yang sedikit berbeda.

| Fitur | XAMPP | Laragon |
| --- | --- | --- |
| **Popularitas** | Sangat legendaris, digunakan sejak dulu, komunitas besar. | Populer di kalangan modern web developer (terutama pengguna Laravel). |
| **Kecepatan** | Standar. | Terkenal sangat cepat dan ringan (*isolated*). |
| **Fitur Unggulan** | Stabil, lintas platform (Windows, Mac, Linux). | *Pretty URLs* (misal: `mysite.test`), mudah ganti versi PHP, *Auto Virtual Hosts*. |
| **Sistem Operasi** | Windows, Linux, macOS. | Khusus Windows. |

---

## 3. Panduan Penggunaan XAMPP

XAMPP adalah singkatan dari **X** (Cross-platform), **A**pache (Web server), **M**ariaDB/MySQL (Database), **P**HP (Bahasa pemrograman), dan **P**erl.

### Cara Kerja di XAMPP

Untuk membuat website di XAMPP, ikuti langkah-langkah berikut:

1. **Jalankan Control Panel:** Start Service.
Buka aplikasi XAMPP, lalu klik tombol **Start** pada modul **Apache** dan **MySQL**. Jika warnanya berubah menjadi hijau, berarti server lokal sudah aktif.


2. **Lokasi Folder Proyek (htdocs):** Tempat Menyimpan File.
Semua file website Anda harus disimpan di dalam folder khusus bernama `htdocs`.

* Jalur default di Windows: `C:\xampp\htdocs\`
* Buat folder baru di dalam htdocs, misalnya bernama `toko-ku`.


3. **Akses Melalui Browser:** Uji Coba.
Buka browser Anda (Chrome/Edge), lalu ketik URL berikut di address bar:
`localhost/toko-ku/`


---

## 4. Panduan Penggunaan Laragon

Laragon adalah alternatif XAMPP yang jauh lebih modern dan kaya fitur, sering menjadi pilihan utama bagi pengguna sistem operasi Windows saat ini.

### Cara Kerja di Laragon

Laragon membuat proses pengelolaan project menjadi jauh lebih praktis:

1. **Jalankan Laragon:** Start All.
Buka Laragon, lalu klik tombol **Start All** di bagian bawah untuk mengaktifkan Apache/Nginx dan MySQL.


2. **Lokasi Folder Proyek (www):** Tempat Menyimpan File.
Berbeda dengan XAMPP, folder proyek di Laragon diletakkan di dalam folder `www`.

* Jalur default: `C:\laragon\www\`
* Buat folder baru di sana, misalnya `bisnis-ku`.


3. **Fitur Pretty URLs:** Akses Otomatis.
Laragon secara otomatis membuatkan *Virtual Host*. Anda tidak perlu mengetik `localhost/bisnis-ku`, melainkan bisa langsung mengaksesnya di browser dengan mengetik:
`bisnis-ku.test`


---

## 5. Mengelola Database (phpMyAdmin)

Baik di XAMPP maupun Laragon, Anda akan menggunakan **phpMyAdmin** untuk membuat dan mengatur database MySQL melalui tampilan web yang mudah dipahami.

* **Cara Akses:** Buka browser, lalu ketik `localhost/phpmyadmin`
* **Fungsi Utama:** Membuat database baru, membuat tabel, mengimpor database (`.sql`), atau mengekspor data sebelum website di-online-kan ke hosting asli.

---

> 💡 **Rangkuman Singkat:**
> Gunakan **XAMPP** jika Anda mencari dokumentasi/tutorial yang melimpah dan menggunakan Mac/Linux. Gunakan **Laragon** jika Anda pengguna Windows yang menginginkan performa lebih cepat, fitur ganti versi PHP yang instan, dan URL lokal yang rapi (`.test`).
