# 📘 Materi Laravel: Pengenalan Laravel

## 📌 Pengertian Laravel

Laravel adalah framework PHP berbasis **MVC (Model-View-Controller)** yang digunakan untuk membangun aplikasi web secara cepat, terstruktur, dan aman. Laravel menyediakan berbagai fitur bawaan seperti routing, authentication, ORM (Eloquent), dan templating engine (Blade).

---

## 🎯 Tujuan Pembelajaran

Setelah mempelajari materi ini, peserta didik diharapkan mampu:

* Memahami konsep dasar framework Laravel
* Mengetahui kelebihan Laravel dibanding PHP native
* Memahami struktur dasar project Laravel
* Menjalankan project Laravel pertama

---

## 🚀 Kelebihan Laravel

* Sintaks yang elegan dan mudah dipahami
* Menggunakan konsep MVC (rapi & terstruktur)
* Memiliki fitur lengkap (routing, middleware, ORM, dll)
* Mendukung keamanan (CSRF protection, hashing, dll)
* Komunitas besar dan dokumentasi lengkap

---

Berikut adalah penjelasan spesifik dari masing-masing komponen:

1. Routing adalah mekanisme yang menghubungkan URL atau endpoint pada web browser dengan kode logika di dalam aplikasi Anda.
    Fungsi: Menentukan halaman mana yang akan dibuka atau fungsi apa yang akan dijalankan ketika pengguna mengklik URL tertentu (seperti /home atau /profile).

2. MiddlewareMiddleware bertindak sebagai lapisan penyaring atau perantara (filter) antara permintaan masuk (request) dan logika utama aplikasi Anda.
    Fungsi: Memeriksa apakah pengguna memiliki hak akses (autentikasi/login), memvalidasi data yang dikirim, atau memblokir serangan seperti CSRF (Cross-Site Request Forgery). Jika lolos filter, request diteruskan ke proses selanjutnya.

3. ORM (Object-Relational Mapping)ORM, yang pada Laravel disebut sebagai Eloquent ORM, adalah fitur yang memungkinkan Anda mengelola database menggunakan Objek PHP.
    Fungsi: Mengubah tabel database menjadi kode berbasis Object-Oriented Programming (OOP). Anda tidak perlu menulis kode SQL secara manual untuk mengambil, menambah, atau mengubah data.

---

## 🧱 Konsep Dasar Laravel (MVC)

### 1. Model

* Berfungsi untuk mengelola data dan berinteraksi dengan database
* Contoh: mengambil data user dari database

### 2. View

* Berfungsi untuk menampilkan data ke pengguna (UI)
* Menggunakan Blade Template Engine

### 3. Controller

* Menghubungkan Model dan View
* Berisi logika aplikasi

---

## 📂 Struktur Folder Laravel

Berikut struktur dasar project Laravel:

```
laravel-app/
│
├── app/            # Berisi logic aplikasi (Model, Controller)
├── routes/         # Tempat mendefinisikan routing
├── resources/      # View (Blade), CSS, JS
├── public/         # File publik (index.php)
├── database/       # Migration & seeder
├── config/         # Konfigurasi aplikasi
├── storage/        # Penyimpanan file
└── vendor/         # Dependency (Composer)
```

# ⚙️ Instalasi PHP Manual (Untuk Development Lanjutan)

Jika Anda hanya membutuhkan PHP tanpa web server bawaan (misalnya untuk penggunaan **Command Line** atau framework tertentu), Anda dapat melakukan instalasi manual dengan langkah berikut:

---

## 📥 1. Unduh PHP

* Kunjungi situs resmi PHP untuk Windows
* Unduh file dengan versi:

  * **VS16 x64 Thread Safe** (atau versi terbaru yang tersedia)

---

## 📂 2. Ekstrak File

* Buat folder baru di drive C:

  ```
  C:\php
  ```
* Ekstrak seluruh isi file `.zip` yang telah diunduh ke dalam folder tersebut

---

## ⚙️ 3. Konfigurasi `php.ini`

* Masuk ke folder:

  ```
  C:\php
  ```
* Cari file:

  ```
  php.ini-development
  ```
* Salin (copy) file tersebut, lalu ubah namanya menjadi:

  ```
  php.ini
  ```

---

## 🌐 4. Pengaturan Environment Variables

### Langkah-langkah:

1. Tekan tombol **Windows**
2. Cari dan pilih:

   ```
   Environment Variables
   ```
3. Klik:

   ```
   Edit the system environment variables
   ```
4. Klik tombol:

   ```
   Environment Variables
   ```
5. Pada bagian **System variables**, cari:

   ```
   Path
   ```
6. Pilih **Path**, lalu klik **Edit**
7. Klik **New**, kemudian tambahkan:

   ```
   C:\php
   ```
8. Klik **OK** pada semua jendela untuk menyimpan

---

## ✅ 5. Verifikasi Instalasi

Buka **Command Prompt (CMD)**, lalu ketik:

```bash
php -v
```

Jika berhasil, akan muncul versi PHP yang telah terinstall.

# 📘 Praktik Laravel: Website Perkenalan Diri

---

## 📌 Deskripsi

Project ini merupakan website sederhana berbasis Laravel yang digunakan untuk menampilkan informasi perkenalan diri seperti nama, biodata, dan kontak.

---

## 🎯 Tujuan Praktik

* Memahami dasar penggunaan Laravel
* Membuat route dan controller
* Menampilkan data ke view (Blade)
* Membuat tampilan web sederhana

---

## ⚙️ Persyaratan

Pastikan sudah terinstall:

* PHP ≥ 8.x
* Composer
* Laravel
* XAMPP / Laragon

---

## 🚀 Langkah Kerja

### 1. Jalankan Project Laravel

Masuk ke folder project:

```bash
cd nama-project
```

---

### 2. Jalankan Server

```bash
php artisan serve
```

Buka di browser:

```
http://127.0.0.1:8000
```

---

## 🧩 Pembuatan Website Perkenalan Diri

### 3. Membuat Route

Buka file `routes/web.php`, tambahkan:

```php
Route::get('/', function () {
    return view('profil');
});
```

---

### 4. Membuat File View (Blade)

Buat file baru:

```
resources/views/profil.blade.php
```

Isi dengan kode berikut:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Perkenalan Diri</title>
    <style>
        body {
            font-family: Arial;
            background-color: #f4f4f4;
            text-align: center;
        }
        .card {
            background: white;
            padding: 20px;
            margin: 50px auto;
            width: 300px;
            border-radius: 10px;
            box-shadow: 0 0 10px rgba(0,0,0,0.1);
        }
        img {
            width: 100px;
            border-radius: 50%;
        }
    </style>
</head>
<body>

<div class="card">
    <img src="https://via.placeholder.com/100" alt="Foto">
    <h2>Nama Kamu</h2>
    <p>Saya adalah seorang pelajar yang sedang belajar Laravel.</p>
    <p><b>Email:</b> emailkamu@gmail.com</p>
</div>

</body>
</html>
```

---
### 5. Menambahkan foto di tempat

 └── public/
     └── img/
         └── foto.jpg
         
---

### 6. Jalankan dan Lihat Hasil

Buka browser:

```
http://127.0.0.1:8000
```
