## 1. Apa itu Variabel?

Bayangkan variabel sebagai sebuah **kotak penyimpanan** di dalam memori komputer. Kotak ini kita beri label (nama), lalu kita bisa memasukkan sesuatu (data/nilai) ke dalamnya. Nilai di dalam kotak tersebut bisa kita ambil, kita gunakan, atau kita ubah kapan saja saat program berjalan.

---

## 2. Aturan Penulisan Variabel di PHP

PHP memiliki aturan yang cukup santai dibandingkan bahasa pemrograman lain (seperti Java atau C++), tetapi Anda tetap harus mengikuti aturan baku berikut agar kode tidak error:

* **Wajib Diawali Simbol `$`:** Semua variabel di PHP harus dimulai dengan tanda dolar, contoh: `$nama`, `$harga`.
* **Karakter Pertama Setelah `$`:** Harus berupa **huruf** atau **underscore (`_`)**. Tidak boleh diawali dengan angka.
* `$nama1` (Benar)
* `$_lokasi` (Benar)
* `$1nama` (Salah/Error)


* **Hanya Boleh Berisi Alfanumerik dan Underscore:** Nama variabel hanya boleh terdiri dari huruf (a-z, A-Z), angka (0-9), dan garis bawah (`_`). Tidak boleh ada spasi atau simbol lain seperti `-`, `@`, `*`.
* `$nama_lengkap` (Benar)
* `$nama lengkap` (Salah karena ada spasi)


* **Sensitif terhadap Huruf Besar/Kecil (*Case-Sensitive*):** Huruf besar dan kecil dianggap berbeda.
* `$produk` dan `$Produk` adalah dua kotak penyimpanan yang **berbeda**.



---

## 3. Cara Membuat dan Mengisi Variabel (Sintaks)

Di PHP, Anda tidak perlu mendeklarasikan tipe data (seperti menentukan bahwa kotak ini hanya boleh diisi angka). PHP secara otomatis akan tahu jenis data apa yang Anda masukkan. Proses mengisi nilai ke dalam variabel menggunakan tanda sama dengan (`=`).

Berikut adalah contoh penulisan di dalam file PHP Anda (misal ditaruh di `C:\laragon\www\belajar\index.php`):

```php
<?php

// Membuat variabel dan mengisinya dengan data
$nama_depan  = "Budi";       // Tipe data Teks (String) -> wajib pakai tanda kutip
$umur        = 22;           // Tipe data Angka Bulat (Integer)
$tinggi_badan = 172.5;       // Tipe data Angka Desimal (Float)
$sudah_menikah = false;      // Tipe data Benar/Salah (Boolean)

// Menampilkan isi variabel ke layar browser
echo "Nama saya adalah: " . $nama_depan;
echo "<br>"; // Pindah baris di browser
echo "Umur saya: " . $umur . " tahun";

?>

```

> 💡 **Mengenal Tanda Titik (`.`):** Tanda titik di atas digunakan untuk **menggabungkan** teks biasa dengan variabel. Di dalam pemrograman, proses penggabungan teks ini disebut *concatenation*.

---

## 4. Keunggulan Variabel: Nilainya Bisa Diubah

Sesuai namanya (*variable* = berubah-ubah), nilai di dalam kotak tersebut bisa kita timpa dengan nilai yang baru. PHP akan membaca kode dari atas ke bawah, sehingga nilai yang terakhir dimasukkan adalah nilai yang akan digunakan.

```php
<?php

$skor = 10; // Skor awal
$skor = 25; // Nilai skor ditimpa dengan yang baru

echo $skor; // Yang muncul di browser adalah 25, bukan 10

?>

```

---

## 5. Tips Penulisan: Gaya CamelCase vs snake_case

Saat membuat nama variabel yang terdiri dari dua kata atau lebih, developer biasanya menggunakan salah satu dari dua gaya penulisan populer ini agar kode mudah dibaca:

1. **snake_case:** Menggunakan underscore sebagai pemisah kata (paling sering dipakai di PHP standar).
* Contoh: `$total_harga_barang`, `$nama_lengkap`


2. **camelCase:** Kata pertama huruf kecil, kata berikutnya diawali huruf besar (sering dipakai di framework seperti Laravel).
* Contoh: `$totalHargaBarang`, `$namaLengkap`



Pilihlah salah satu gaya yang paling Anda sukai dan gunakan secara konsisten di seluruh proyek Anda.
