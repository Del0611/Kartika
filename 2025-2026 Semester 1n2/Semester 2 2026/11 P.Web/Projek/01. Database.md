## Bagian 1: Membuat Database di Laragon (Lebih Cepat)

Laragon menyediakan akses instan ke alat manajemen database. Anda bisa menggunakan **phpMyAdmin** seperti biasa, atau menggunakan **HeidiSQL** (aplikasi desktop bawaan Laragon yang jauh lebih ringan).

Mari kita gunakan cara cepat via phpMyAdmin:

1. **1. Buka phpMyAdmin lewat Menu Laragon:** Laragon UI.
* Pastikan Laragon sudah dalam kondisi **Start All**.
* **Klik kanan** di area mana saja pada jendela Laragon.
* Pilih menu **MySQL** > **phpMyAdmin**. Browser Anda akan otomatis terbuka ke halaman login.


2. **2. Login phpMyAdmin:** Credentials.
* **Username:** `root`
* **Password:** (Biarkan kosong/tidak usah diisi)
* Klik **Go / Masuk**.


3. **3. Buat Database Baru:** Database Base.
* Klik menu **New** atau **Baru** di panel sebelah kiri.
* Masukkan nama database Anda, misalnya: `db_laragon`.
* Klik tombol **Create / Buat**. Wadah data Anda sudah siap!

## Bagian 2: Menulis Script Koneksi PHP di Laragon

Sekarang kita akan memanfaatkan fitur *Pretty URLs* (`.test`) milik Laragon agar pengujian kode menjadi lebih profesional.

### 1. Letakkan File di Folder `www`

Buka text editor Anda (VS Code), lalu buka folder proyek Anda yang berada di dalam Laragon.

* Jalur folder: `C:\laragon\www\nama-proyek-anda\`
* Buat file baru di sana dengan nama **`koneksi.php`**.

### 2. Tulis Kode Koneksi (Menggunakan PDO)

Salin kode PHP standar industri di bawah ini ke dalam file `koneksi.php`:

```php
<?php
// Konfigurasi database untuk Laragon
$host     = "localhost";
$username = "root";         // Default user Laragon
$password = "";             // Default password Laragon (kosong)
$database = "db_laragon";   // Nama database yang baru saja kita buat

try {
    // Membuat koneksi dengan PDO
    $koneksi = new PDO("mysql:host=$host;dbname=$database", $username, $password);
    
    // Mengaktifkan mode error reporting untuk mempermudah tracking bug
    $koneksi->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    
    echo "Koneksi database via Laragon BERHASIL!"; 
} catch (PDOException $e) {
    // Menampilkan pesan jika ada kegagalan koneksi
    echo "Koneksi GAGAL: " . $e->getMessage();
}
?>

```

### 3. Cara Menguji di Browser

Inilah kelebihan Laragon. Anda tidak perlu mengetik `localhost/nama-proyek/...`. Cukup buka browser dan ketik URL sesuai nama folder Anda diikuti `.test`:

```text
http://nama-proyek-anda.test/koneksi.php

```

> ⚠️ **Catatan penting pengguna Laragon:** Jika saat membuka `.test` muncul error atau halaman tidak ditemukan, pastikan Anda sudah melakukan **Stop** lalu **Start All** kembali pada aplikasi Laragon Anda. Proses ini diperlukan agar Laragon bisa mendaftarkan nama folder baru Anda ke dalam sistem pengalihan URL otomatisnya (*Auto Virtual Hosts*).
