# 📘 Materi Python: Functions (Fungsi)

## 📌 Pengertian Function
Function (fungsi) adalah blok kode yang digunakan untuk menjalankan tugas tertentu dan dapat dipanggil kembali kapan saja. Fungsi membantu membuat program lebih rapi, terstruktur, dan mudah dipahami.

---

## 🎯 Tujuan Penggunaan Function
- Menghindari penulisan kode berulang
- Membuat program lebih modular
- Mempermudah debugging (perbaikan error)
- Meningkatkan keterbacaan kode

---

## 🧱 Struktur Dasar Function

```python
def nama_fungsi(parameter):
    # kode yang dijalankan
    return nilai
````

### Penjelasan:

* `def` → kata kunci untuk membuat fungsi
* `nama_fungsi` → nama fungsi
* `parameter` → input (opsional)
* `return` → mengembalikan nilai (opsional)

---

## ✨ Contoh Function Sederhana

```python
def sapa():
    print("Halo, selamat belajar Python!")
    
sapa()
```

---

## 📥 Function dengan Parameter

```python
def sapa_nama(nama):
    print("Halo", nama)

sapa_nama("Andi")
```

---

## 📤 Function dengan Return Value

```python
def tambah(a, b):
    return a + b

hasil = tambah(5, 3)
print("Hasil:", hasil)
```

---

## 🔄 Function dengan Default Parameter

```python
def sapa(nama="User"):
    print("Halo", nama)

sapa()
sapa("Budi")
```

---

## 📦 Function dengan Banyak Parameter

```python
def hitung_luas(panjang, lebar):
    luas = panjang * lebar
    return luas

print(hitung_luas(5, 4))
```

---

## 🔁 Function Rekursif

Fungsi yang memanggil dirinya sendiri.

```python
def faktorial(n):
    if n == 1:
        return 1
    else:
        return n * faktorial(n-1)

print(faktorial(5))
```

---

## 🧠 Jenis-Jenis Function

1. **Built-in Function** → fungsi bawaan Python (contoh: `print()`, `len()`)
2. **User-defined Function** → fungsi yang dibuat sendiri
3. **Lambda Function** → fungsi tanpa nama

---

## ⚡ Contoh Lambda Function

```python
tambah = lambda a, b: a + b
print(tambah(3, 2))
```

---

## 🧪 Latihan Praktik

### 1. Buat function untuk menghitung keliling persegi!

```python
def keliling_persegi(sisi):
    return 4 * sisi
```

### 2. Buat function untuk mengecek bilangan genap/ganjil!

```python
def cek_bilangan(angka):
    if angka % 2 == 0:
        return "Genap"
    else:
        return "Ganjil"
```

---

## 📌 Kesimpulan

Function adalah bagian penting dalam Python yang digunakan untuk mengelompokkan kode agar lebih efisien, mudah digunakan ulang, dan terstruktur dengan baik.

---

## 🚀 Tips

* Gunakan nama fungsi yang jelas dan deskriptif
* Hindari fungsi terlalu panjang
* Gunakan `return` jika ingin menghasilkan nilai

# 📦 Modularisasi Python

## 📖 Pengertian

Modularisasi dalam Python adalah teknik memecah program besar menjadi bagian-bagian kecil yang disebut **module** agar kode lebih terstruktur, mudah dibaca, dan mudah dipelihara.

---

## 🎯 Tujuan Modularisasi

* Mempermudah pengembangan program
* Mengurangi duplikasi kode
* Memudahkan debugging (perbaikan error)
* Meningkatkan keterbacaan kode
* Memungkinkan penggunaan ulang (reusable code)

---

## 🧱 Jenis Modularisasi di Python

### 1. Module

File Python (`.py`) yang berisi fungsi, class, atau variabel.

**Contoh:**

```python
# file: matematika.py
def tambah(a, b):
    return a + b

def kurang(a, b):
    return a - b
```

---

### 2. Package

Folder yang berisi beberapa module dan memiliki file `__init__.py`.

**Struktur:**

```
kalkulator/
│
├── __init__.py
├── operasi.py
└── lanjutan.py
```

---

## 📥 Cara Menggunakan Module

### 1. Import seluruh module

```python
import matematika

print(matematika.tambah(5, 3))
```

---

### 2. Import fungsi tertentu

```python
from matematika import tambah

print(tambah(10, 2))
```

---

### 3. Import dengan alias

```python
import matematika as m

print(m.kurang(8, 3))
```

---

### 4. Import semua isi module (tidak disarankan)

```python
from matematika import *

print(tambah(1, 2))
```

---

## 📂 Module Built-in Python

Python sudah menyediakan module bawaan seperti:

* `math` → operasi matematika
* `random` → angka acak
* `datetime` → tanggal & waktu
* `os` → sistem operasi

**Contoh:**

```python
import math

print(math.sqrt(25))
```

---

## 🛠️ Membuat Module Sendiri

1. Buat file `.py`
2. Isi dengan fungsi atau class
3. Import di file lain

---

## ⚙️ Contoh Program Modular

### File: operasi.py

```python
def kali(a, b):
    return a * b

def bagi(a, b):
    return a / b
```

---

### File: main.py

```python
import operasi

print("Perkalian:", operasi.kali(4, 5))
print("Pembagian:", operasi.bagi(10, 2))
```

---

## 🚫 Kesalahan Umum

* Lupa menambahkan `.py`
* Salah penulisan nama module
* Tidak satu folder saat import
* Menggunakan `import *` secara berlebihan

---

## 💡 Tips

* Gunakan nama module yang jelas
* Pisahkan fungsi sesuai tugasnya
* Gunakan package untuk proyek besar

---

## 📝 Kesimpulan

Modularisasi Python membantu membuat program lebih rapi, efisien, dan mudah dikembangkan. Dengan membagi kode menjadi module dan package, programmer dapat bekerja lebih terstruktur dan profesional.

# 💻 Proyek Mini: Aplikasi Kalkulator Modular

## 🎯 Tujuan

Membuat aplikasi kalkulator sederhana menggunakan konsep modularisasi.

---

## 📂 Struktur Folder

```bash
kalkulator/
│
├── operasi.py
├── menu.py
└── main.py
```

---

## 🧱 1. File: operasi.py

```python id="nyq1fy"
def tambah(a, b):
    return a + b

def kurang(a, b):
    return a - b

def kali(a, b):
    return a * b

def bagi(a, b):
    if b == 0:
        return "Tidak bisa dibagi 0"
    return a / b
```

---

## 📋 2. File: menu.py

```python id="7d8k4w"
def tampilkan_menu():
    print("\n=== KALKULATOR SEDERHANA ===")
    print("1. Penjumlahan")
    print("2. Pengurangan")
    print("3. Perkalian")
    print("4. Pembagian")
    print("5. Keluar")
```

---

## 🚀 3. File: main.py

```python id="0l9d2g"
import operasi
import menu

while True:
    menu.tampilkan_menu()
    pilihan = input("Pilih menu (1-5): ")

    if pilihan == "5":
        print("Terima kasih!")
        break

    a = float(input("Masukkan angka pertama: "))
    b = float(input("Masukkan angka kedua: "))

    if pilihan == "1":
        print("Hasil:", operasi.tambah(a, b))
    elif pilihan == "2":
        print("Hasil:", operasi.kurang(a, b))
    elif pilihan == "3":
        print("Hasil:", operasi.kali(a, b))
    elif pilihan == "4":
        print("Hasil:", operasi.bagi(a, b))
    else:
        print("Pilihan tidak valid!")
```

---

## ▶️ Cara Menjalankan

1. Simpan semua file dalam satu folder
2. Buka terminal / CMD
3. Jalankan:

```bash
python main.py
```
