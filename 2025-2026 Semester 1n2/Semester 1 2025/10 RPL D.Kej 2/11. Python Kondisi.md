# 📘 MATERI PENJELASAN

## Struktur Kontrol Percabangan & Perulangan (Python)

## 1. 🔀 Percabangan (Conditional Statement)

Percabangan digunakan untuk **mengambil keputusan** berdasarkan kondisi tertentu.

### 🔹 a. `if`

Digunakan untuk menjalankan kode jika kondisi benar.

```python
nilai = 80

if nilai >= 75:
    print("Lulus")
```

---

### 🔹 b. `if - else`

Digunakan jika ada dua kondisi (benar atau salah).

```python
nilai = 60

if nilai >= 75:
    print("Lulus")
else:
    print("Tidak Lulus")
```

---

### 🔹 c. `if - elif - else`

Digunakan untuk banyak kondisi.

```python
nilai = 85

if nilai >= 90:
    print("A")
elif nilai >= 80:
    print("B")
elif nilai >= 70:
    print("C")
else:
    print("D")
```

---

### 🔹 d. Operator Perbandingan

Digunakan dalam percabangan:

| Operator | Arti             |
| -------- | ---------------- |
| `==`     | Sama dengan      |
| `!=`     | Tidak sama       |
| `>`      | Lebih besar      |
| `<`      | Lebih kecil      |
| `>=`     | Lebih besar sama |
| `<=`     | Lebih kecil sama |

---

### 🔹 e. Operator Logika

Digunakan untuk kondisi lebih kompleks:

* `and` → semua kondisi harus benar
* `or` → salah satu benar
* `not` → kebalikan

Contoh:

```python
nilai = 80
absen = 90

if nilai >= 75 and absen >= 80:
    print("Lulus")
```

---

## 2. 🔁 Perulangan (Looping)

Perulangan digunakan untuk menjalankan kode **berulang kali**.

---

### 🔹 a. `for`

Digunakan untuk perulangan dengan jumlah pasti.

```python
for i in range(5):
    print("Perulangan ke-", i)
```

📌 Output:

```
Perulangan ke- 0
Perulangan ke- 1
...
```

---

### 🔹 b. `while`

Digunakan selama kondisi masih benar.

```python
i = 1

while i <= 5:
    print("Angka:", i)
    i += 1
```

---

### 🔹 c. `break`

Menghentikan perulangan.

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

---

### 🔹 d. `continue`

Melewati satu iterasi.

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

---

### 🔹 e. `pass`

Tidak melakukan apa-apa (placeholder).

```python
for i in range(5):
    pass
```

# 🧪 MATERI PRAKTIK (LATIHAN)

## 📝 Praktik 1: Menentukan Bilangan Ganjil/Genap

```python
angka = int(input("Masukkan angka: "))

if angka % 2 == 0:
    print("Genap")
else:
    print("Ganjil")
```

---

## 📝 Praktik 2: Program Nilai Siswa

```python

if nilai >= 90:
    print("A")
elif nilai >= 80:
    print("B")
elif nilai >= 70:
    print("C")
else:
    print("D")
```

---

## 📝 Praktik 3: Menampilkan Angka 1–10

```python
for i in range(1, 11):
    print(i)
```

---

## 📝 Praktik 4: Menjumlahkan Angka

```python
total = 0

for i in range(1, 6):
    total += i

print("Total:", total)
```

---

## 📝 Praktik 5: Tebak Angka (Loop + Percabangan)

```python
angka_rahasia = 7
tebakan = 0

while tebakan != angka_rahasia:
    tebakan = int(input("Tebak angka (1-10): "))
    
    if tebakan == angka_rahasia:
        print("Benar!")
    else:
        print("Salah, coba lagi")
```

---

## 📝 Praktik 6: Menu Sederhana

```python
while True:
    print("\nMenu:")
    print("1. Halo")
    print("2. Keluar")
    
    pilih = input("Pilih menu: ")
    
    if pilih == "1":
        print("Halo User!")
    elif pilih == "2":
        print("Keluar...")
        break
    else:
        print("Pilihan salah")
```
