## 📦 BAGIAN 1: Variabel, Tipe Data, dan Operator

### 1. Variabel 📝
**Variabel** adalah sebuah wadah di dalam memori komputer yang digunakan untuk menyimpan nilai atau data selama program berjalan. Nilai di dalam variabel bersifat dinamis (bisa diubah-ubah).

Di Java, sebelum menggunakan variabel, Anda **wajib mendeklarasikannya** terlebih dahulu dengan menentukan tipe datanya.

* **Sintaks Dasar:**
  ```java
  tipe_data nama_variabel = nilai_awal;
  

```

* **Contoh Kode:**
```java
int skor = 100;       // Mengisi wadah bernama 'skor' dengan angka bulat 100
String nama = "Aris"; // Mengisi wadah bernama 'nama' dengan teks "Aris"


```



---

### 2. Tipe Data 🗂️
Java adalah bahasa yang *strongly typed*, artinya setiap variabel harus jelas jenis data apa yang bisa ditampungnya. Tipe data di Java dibagi menjadi dua kategori utama:

#### A. Tipe Data Primitif (Bawaan Sistem)
* 🔢 **`int`**: Untuk angka bulat tanpa desimal (contoh: `10`, `-50`, `2026`).
* 📐 **`double`**: Untuk angka pecahan atau desimal (contoh: `3.14`, `0.005`).
* 🔤 **`char`**: Untuk satu karakter tunggal, wajib diapit petik tunggal (contoh: `'A'`, `'7'`, `'%'`).
* ⚖️ **`boolean`**: Hanya memiliki dua nilai logika, yaitu `true` (benar) atau `false` (salah).

#### B. Tipe Data Referensi / Non-Primitif
* 📄 **`String`**: Kumpulan dari banyak karakter atau teks, wajib diapit petik ganda (contoh: `"Halo Bandung"`).

| Tipe Data | Kategori | Contoh Nilai | Contoh Kode Java |
| :---: | :--- | :--- | :--- |
| `int` | Primitif | `25` | `int umur = 25;` |
| `double` | Primitif | `3.14` | `double pi = 3.14;` |
| `char` | Primitif | `'B'` | `char kelas = 'B';` |
| `boolean`| Primitif | `true` | `boolean isActive = true;` |
| `String` | Referensi| `"Java"` | `String bahasa = "Java";` |

---

### 3. Operator ⚡
Operator adalah simbol khusus yang digunakan untuk melakukan manipulasi atau operasi terhadap variabel dan nilai.

#### A. ➕ Operator Aritmatika (Perhitungan)
* `+` (Penjumlahan) $\rightarrow$ `5 + 3 = 8`
* `-` (Pengurangan) $\rightarrow$ `5 - 3 = 2`
* `*` (Perkalian) $\rightarrow$ `5 * 3 = 15`
* `/` (Pembagian) $\rightarrow$ `10 / 2 = 5`
* `%` (Sisa Bagi / Modulo) $\rightarrow$ `10 % 3 = 1` *(karena 10 dibagi 3 sisa 1)*

#### B. 🔍 Operator Relasional (Perbandingan)
Hasil dari operator ini selalu berwujud nilai boolean (`true`/`false`).
* `==` (Sama dengan)
* `!=` (Tidak sama dengan)
* `>`  (Lebih besar dari)
* `<`  (Lebih kecil dari)
* `>=` (Lebih besar atau sama dengan)
* `<=` (Lebih kecil atau sama dengan)

#### C. 🧠 Operator Logika
Digunakan untuk menghubungkan dua atau lebih kondisi boolean.
* `&&` (**DAN / AND**): Menghasilkan `true` jika **semua** kondisi bernilai benar.
* `||` (**ATAU / OR**): Menghasilkan `true` jika **salah satu** kondisi bernilai benar.
* `!`  (**TIDAK / NOT**): Membalikkan nilai logika (jika `true` diubah menjadi `false`).

## 🎛️ BAGIAN 2: Struktur Kontrol

Secara default, kode berjalan lurus dari atas ke bawah. Struktur kontrol bertindak layaknya polisi lalu lintas yang mengatur alur jalannya program, apakah harus berbelok (percabangan) atau berputar (perulangan).

### 1. Percabangan (Decision Making) 🚦
Memungkinkan program mengambil keputusan dan mengeksekusi blok kode tertentu hanya jika kondisi yang disyaratkan terpenuhi.

#### 🟢 Struktur `if` (Satu Pilihan)
```java
int nilai = 80;
if (nilai >= 75) {
    System.out.println("Anda Lulus!");
}

```

#### 🟡 Struktur `if - else` (Dua Pilihan)

```java
int umur = 16;
if (umur >= 17) {
    System.out.println("Bisa membuat SIM");
} else {
    System.out.println("Belum cukup umur untuk membuat SIM");
}

```

#### 🔴 Struktur `if - else if - else` (Banyak Pilihan)

```java
int skorUjian = 85;

if (skorUjian >= 90) {
    System.out.println("Nilai: A");
} else if (skorUjian >= 80) {
    System.out.println("Nilai: B");
} else {
    System.out.println("Nilai: C");
}

```

#### 🔀 Struktur `switch - case` (Percabangan Nilai Konstan)

Digunakan sebagai alternatif `if - else if` ketika kondisi yang diperiksa berupa nilai konstan/pasti (seperti angka bulat, karakter, atau String). Struktur ini mencocokkan nilai variabel secara langsung tanpa operasi logika rumit.

* **Sintaks:**
```java
switch (variabel) {
    case nilai1:
        // kode jika variabel == nilai1
        break;
    case nilai2:
        // kode jika variabel == nilai2
        break;
    default:
        // kode jika tidak ada nilai yang cocok
}

```


* **Contoh Kode:**
```java
int pilihanMenu = 2;

switch (pilihanMenu) {
    case 1:
        System.out.println("Membuka Menu Profil 👤");
        break;
    case 2:
        System.out.println("Membuka Menu Pengaturan ⚙️");
        break;
    case 3:
        System.out.println("Keluar Aplikasi ❌");
        break;
    default:
        System.out.println("Pilihan tidak valid! ⚠️");
}


```


* 💡 **Catatan Penting:** Kata kunci `break` wajib ditulis di akhir setiap `case`. Jika lupa, program akan terus mengeksekusi *case* di bawahnya secara berurutan meskipun nilainya tidak cocok (*fall-through*).

#### 🔀 Operator Ternary (Percabangan Singkat)
Bentuk ringkas dari struktur `if - else` yang mengembalikan sebuah nilai. Sangat berguna untuk mempersingkat baris kode saat melakukan penugasan variabel berdasarkan kondisi tertentu.

* **Sintaks:**
  ```java
  tipe_data variabel = (kondisi) ? nilai_jika_true : nilai_jika_false;
  

```

* **Contoh Kode:**
```java
int nilaiSiswa = 78;
// Jika nilaiSiswa >= 75 maka "Lulus", jika kurang maka "Remidi"
String status = (nilaiSiswa >= 75) ? "Lulus" : "Remidi";

System.out.println("Status Siswa: " + status);

```

### 2. Perulangan (Looping) 🔄

Perulangan digunakan untuk menjalankan baris kode yang sama berulang kali selama kondisi bernilai `true`.

#### 🔁 Perulangan `for`

Paling ideal digunakan jika kita **sudah tahu pasti** berapa kali perulangan tersebut harus berjalan.

```java
// Mencetak angka 1 sampai 3
for (int i = 1; i <= 3; i++) {
    System.out.println("Perulangan ke-" + i);
}

```

*💡 **Cara membaca:*** Mulai dari angka `i = 1`, cek apakah `i <= 3`? Jika iya, jalankan perintah cetak, lalu naikkan nilai `i` sebanyak 1 tingkat (`i++`). Ulangi terus hingga kondisi bernilai salah.

#### 🔁 Perulangan `while`

Digunakan ketika jumlah perulangannya **belum diketahui pasti**, karena perulangan ini murni bergantung pada suatu kondisi yang dinamis.

```java
int energi = 3;
while (energi > 0) {
    System.out.println("Robot berjalan... 🤖");
    energi--; // Mengurangi energi sebesar 1 setiap melangkah
}

```

## 🛠️ Studi Kasus: Program Gabungan

Mari satukan seluruh konsep di atas ke dalam satu skenario dunia nyata:

```java
public class AplikasiKelulusan {
    public static void main(String[] args) {
        // 1. Inisialisasi Data (Variabel & Tipe Data)
        String namaSiswa = "Budi";
        int nilaiTugas = 85;
        boolean hadirUjian = true;
        
        // 2. Logika Keputusan (Operator & Percabangan)
        // Syarat lulus: Harus hadir ujian DAN nilai tugas wajib di atas 70
        if (hadirUjian && nilaiTugas > 70) {
            System.out.println("🎉 Selamat " + namaSiswa + ", Anda dinyatakan LULUS!");
            
            // 3. Eksekusi Proses Berulang (Perulangan)
            System.out.println("\nMemproses pencetakan sertifikat penghargaan:");
            for (int i = 1; i <= 3; i++) {
                System.out.println("🖨️ Mencetak salinan sertifikat ke-" + i);
            }
        } else {
            System.out.println("❌ Maaf " + namaSiswa + ", Anda TIDAK LULUS.");
        }
    }
}

```
