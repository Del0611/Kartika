## 🏗️ 1. Swing Containers
*Containers* adalah komponen yang berfungsi sebagai wadah untuk menampung dan mengorganisir komponen lainnya.

| No | Nama Komponen | Deskripsi Fungsi |
| :--- | :--- | :--- |
| 1 | **Panel** | Digunakan untuk mengelompokkan komponen-komponen agar tata letak lebih rapi. |
| 2 | **Split Pane** | Menampilkan dua komponen dalam ruang tetap yang ukurannya dapat digeser oleh pengguna. |
| 3 | **Tool Bar** | Tempat menampilkan alat (tool) yang sering digunakan untuk mempercepat akses operasional. |
| 4 | **Internal Frame** | Frame yang berada di dalam frame utama (memungkinkan konsep *multi-window* dalam satu aplikasi). |
| 5 | **Tabbed Pane** | Membuat tampilan berlapis menggunakan tab yang bisa diklik oleh pengguna. |
| 6 | **Scroll Pane** | Menyediakan batang penggulung (*scroll bar*) otomatis jika komponen di dalamnya melebihi ukuran layar. |
| 7 | **Desktop Pane** | Wadah khusus untuk mengelola banyak *Internal Frame*. |
| 8 | **Layered Pane** | Menyediakan kedalaman (tiga dimensi) untuk mengatur posisi komponen secara bertumpuk. |

## 🎮 2. Swing Controls
*Controls* adalah komponen interaktif yang digunakan pengguna untuk memasukkan data atau melihat informasi.

| No | Nama Komponen | Deskripsi Fungsi |
| :--- | :--- | :--- |
| 1 | **Label** | Menampilkan teks statis sebagai keterangan atau identitas komponen lain. |
| 2 | **Button** | Tombol standar berbentuk persegi panjang dengan tulisan di tengahnya. |
| 3 | **Toggle Button** | Tombol yang memiliki dua kondisi (on/off). Klik pertama mengaktifkan, klik kedua mematikan. |
| 4 | **Radio Button** | Digunakan untuk memilih **satu** dari beberapa pilihan (misal: Jenis Kelamin). |
| 5 | **Check Box** | Digunakan untuk memilih **lebih dari satu** pilihan dari daftar yang tersedia. |
| 6 | **Combo Box** | Daftar pilihan *drop-down* yang menghemat tempat karena pilihan disembunyikan sebelum diklik. |
| 7 | **List** | Menampilkan beberapa item dalam bentuk daftar yang bisa dipilih. |
| 8 | **Text Field** | Kolom satu baris untuk menginput atau menampilkan teks dan angka (misal: hasil kalkulator). |
| 9 | **Password Field** | Kolom input teks yang karakternya disamarkan untuk keamanan kata sandi. |
| 10 | **Text Area** | Kolom input teks banyak baris yang biasanya dilengkapi dengan fungsi scroll. |
| 11 | **Scroll Bar** | Batang penggeser untuk menavigasi jendela secara vertikal atau horizontal. |
| 12 | **Slider** | Visualisasi proses perjalanan (seperti volume suara atau durasi video). |
| 13 | **Progress Bar** | Menampilkan status persentase dari sebuah proses yang sedang berjalan. |
| 14 | **Spinner** | Input angka ringkas dengan panah atas/bawah untuk menaikkan atau menurunkan nilai. |
| 15 | **Separator** | Garis pemisah atau "sekat" antar bagian dalam aplikasi. |
| 16 | **Table** | Menampilkan data dalam format baris dan kolom. |
| 17 | **Tree** | Menampilkan data dalam bentuk hirarki atau struktur pohon (seperti folder). |
| 18 | **Button Group** | Menyatukan fungsi beberapa button agar bekerja sebagai satu kesatuan (penting untuk Radio Button). |
| 19 | **Editor Pane** | Panel yang digunakan untuk menampilkan dan mengedit dokumen teks. |
| 20 | **Text Pane** | Mirip Editor Pane, namun mendukung gaya teks yang lebih kompleks (seperti cetak tebal/miring). |
| 21 | **Formatted Field** | Text field khusus yang membatasi input sesuai format tertentu (misal: format tanggal/uang). |

## 📑 3. Swing Menus
*Menus* digunakan untuk mengatur perintah-perintah aplikasi agar terorganisir di bagian atas jendela.

| No | Nama Komponen | Deskripsi Fungsi |
| :--- | :--- | :--- |
| 1 | **Menu Bar** | Baris utama di bagian atas jendela yang menampung tab-tab menu utama. |
| 2 | **Menu** | Induk perintah yang jika diklik akan memunculkan daftar perintah yang lebih spesifik. |
| 3 | **Menu Item** | Perintah spesifik yang berada di dalam sebuah Menu. |
| 4 | **CheckBox Menu Item** | Menu item yang memiliki tanda centang ($\checkmark$) untuk mengaktifkan/mematikan fitur. |
| 5 | **RadioButton Menu Item** | Menu item untuk memilih satu opsi di dalam kategori menu tertentu. |

| 7 | **Separator** | Garis tipis pemisah antar kelompok perintah di dalam sebuah menu. |

## 🧩 1. Komponen Dasar Program Java di NetBeans

Setiap program Java memiliki **struktur dasar (komponen utama)** agar bisa dikenali oleh komputer.
Dalam NetBeans, struktur program Java biasanya terdiri dari:

### 🧠 a. Header (Bagian Atas Program)

Bagian **header** berisi:

* **Pernyataan paket (package)** — menunjukkan lokasi file dalam proyek.
* **Import library** — digunakan untuk menambahkan fungsi tambahan dari Java Library.

📄 **Contoh Header:**

```java
package myfirstproject;       // Menunjukkan nama paket
import java.util.Scanner;     // Mengimpor library untuk input data
```

> 💡 `package` bersifat opsional jika proyek tidak menggunakan struktur folder kompleks.
> 💡 `import` hanya digunakan jika kamu membutuhkan fitur tambahan seperti input, tanggal, atau database.

---

### ⚙️ b. Body (Isi Utama Program)

Bagian **body** adalah bagian inti tempat logika program ditulis.
Biasanya terdiri dari:

1. **Deklarasi kelas (class)**
2. **Metode utama (main method)**
3. **Isi kode / logika program**

📄 **Contoh Body:**

```java
public class HelloWorld {             // Deklarasi kelas utama
    public static void main(String[] args) {    // Metode utama
        System.out.println("Hello, NetBeans!"); // Baris program (output)
    }
}
```

🧩 **Penjelasan:**

* `public class HelloWorld` → mendefinisikan sebuah kelas bernama *HelloWorld*.
* `public static void main(String[] args)` → titik awal program Java dijalankan.
* `System.out.println()` → perintah untuk menampilkan teks ke layar.

> ⚠️ Setiap program Java **harus memiliki** metode `main()` agar bisa dijalankan.

---

### 📘 c. Footer (Bagian Penutup Program)

Bagian **footer** berisi tanda kurung kurawal penutup `{}` yang menandai akhir dari kelas atau metode.
Java sangat ketat terhadap pembukaan `{` dan penutupan `}`.

📄 **Contoh Footer:**

```java
    } // Menutup metode main
}     // Menutup kelas HelloWorld
```

> 💡 Pastikan jumlah `{` dan `}` seimbang, jika tidak maka program akan error saat dijalankan.

---

### 📂 Struktur Lengkap Program Java di NetBeans

Berikut contoh lengkap tampilan program di NetBeans:

```java
package myfirstproject;      // HEADER

public class HelloWorld {    // BODY
    public static void main(String[] args) {
        System.out.println("Hello, NetBeans!");
    }
}                            // FOOTER
```

📦 **Tampilan di Project Explorer NetBeans:**

```
MyFirstProject
 └── Source Packages
     └── myfirstproject
         └── HelloWorld.java
```

## 🔠 2. Sintaks Dasar Bahasa Pemrograman Java

### 🧩 a. Komentar (Comment)

Komentar digunakan untuk memberikan catatan di dalam kode, tidak dijalankan oleh komputer.

```java
// Komentar satu baris
/* Komentar
   lebih dari satu baris */
```

---

### 🔢 b. Variabel dan Tipe Data

Variabel digunakan untuk menyimpan nilai (angka, teks, dsb).

📄 **Contoh:**

```java
int umur = 18;              // Menyimpan angka bulat
double tinggi = 170.5;      // Menyimpan angka desimal
String nama = "Ayu";        // Menyimpan teks
boolean aktif = true;       // Menyimpan nilai benar/salah
```

🧠 **Tipe Data Umum di Java:**

| Tipe Data | Deskripsi                | Contoh Nilai      |
| :-------- | :----------------------- | :---------------- |
| `int`     | Bilangan bulat           | `10`, `-5`, `100` |
| `double`  | Bilangan desimal         | `3.14`, `25.6`    |
| `char`    | Karakter tunggal         | `'A'`, `'b'`      |
| `String`  | Kumpulan karakter (teks) | `"Hello"`         |
| `boolean` | Nilai logika             | `true`, `false`   |

---

### 🔁 c. Operator Aritmatika

Operator digunakan untuk melakukan perhitungan.

| Operator | Fungsi      | Contoh  |
| :------- | :---------- | :------ |
| `+`      | Penjumlahan | `a + b` |
| `-`      | Pengurangan | `a - b` |
| `*`      | Perkalian   | `a * b` |
| `/`      | Pembagian   | `a / b` |
| `%`      | Sisa bagi   | `a % b` |

📄 **Contoh:**

```java
int a = 10, b = 3;
System.out.println(a + b); // Output: 13
System.out.println(a % b); // Output: 1
```

---

### 🔣 d. Struktur Kontrol (Percabangan)

Digunakan untuk membuat keputusan.

📄 **Contoh If-Else:**

```java
int nilai = 80;

if (nilai >= 75) {
    System.out.println("Lulus");
} else {
    System.out.println("Tidak Lulus");
}
```

📄 **Contoh Switch:**

```java
int hari = 2;

switch (hari) {
    case 1: System.out.println("Senin"); break;
    case 2: System.out.println("Selasa"); break;
    default: System.out.println("Hari lain");
}
```

---

### 🔁 e. Perulangan (Loop)

Digunakan untuk mengulang perintah beberapa kali.

📄 **Contoh For Loop:**

```java
for (int i = 1; i <= 5; i++) {
    System.out.println("Perulangan ke-" + i);
}
```

📄 **Contoh While Loop:**

```java
int i = 1;
while (i <= 3) {
    System.out.println("Angka: " + i);
    i++;
}
```

---

### 📥 f. Input dari Pengguna

Gunakan class `Scanner` untuk menerima input dari keyboard.

📄 **Contoh:**

```java
import java.util.Scanner;

public class InputContoh {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        System.out.print("Masukkan nama: ");
        String nama = input.nextLine();
        
        System.out.println("Halo, " + nama + "!");
    }
}
```

> 💡 Jangan lupa menambahkan `import java.util.Scanner;` di bagian header untuk menggunakan fitur input.

---

### 📤 g. Output ke Layar

Gunakan perintah:

```java
System.out.println("Teks yang ingin ditampilkan");
```

* `println` = menampilkan teks dan ganti baris baru
* `print` = menampilkan teks tanpa pindah baris

---

## 🧭 Kesimpulan

Setiap program Java memiliki **tiga bagian utama:**

1. 🧩 **Header** → deklarasi paket dan import library
2. ⚙️ **Body** → logika utama program (class dan method)
3. 📘 **Footer** → penutup kurung kurawal

Selain itu, kamu juga telah mempelajari **sintaks dasar Java**, seperti:

* Variabel dan tipe data
* Operator
* Percabangan dan perulangan
* Input dan output

> 💬 “Pahami struktur, kuasai sintaks — dan kamu akan bisa membuat program apa pun.”

### 2. Latihan

Coba buat program Java baru di NetBeans dengan langkah berikut:

1. Tambahkan class bernama `Latihan2_Variabel`
2. Program : 
```Java
package latihan;

public class latihan_2_variable {
    public static  void main (String [] args){
        System.out.println ("=== PROGRAM BELAJAR VARIABLE JAVA ===");
        System.out.println();
        
        // 1. Varible primitive (Dasar)
        System.out.println ("1. VARIABEL PRIMITIVE");
        System.out.println("----------------------");
        
        // integer (Bilangan bulat)
        int umur = 20;
        byte nilaiUjian = 85;
        short jumlahSiswa = 150;
        long populasiKota = 5000000L;
        
        System.out. println("umur:" + umur);
        System.out.println ("Nilai Ujian:" + nilaiUjian);
        System.out.println("Jumlah Siswa: " + jumlahSiswa);
        System.out.println("Populasi Kota: " + populasiKota);
        System.out.println();
        
        //Floating point (bilangan desimal)
        float tinggiBadan = 175.5f;
        double hargaBarang = 125000.75;
        
        System.out.println("Tinggi Badan: " + tinggiBadan + " cm");
        System.out.println("Harga Barang: RP " + hargaBarang);
        System.out.println();
        
        // Chracter dan Boolean
        char grade = 'A' ;
        boolean isLulus = true;
        
        System.out.println("Grade: " + grade);
        System.out.println("Status Kelulusan: " + isLulus);
        System.out.println();
        
        // 2. Variabel Reference
        System.out.println("2. VARIABEL RERENCE");
        System.out.println("-------------------");
        
        String nama = "Budi Santoso";
        String alamat = "Jl. Merdeka No. 123";
        
        System.out.println("Nama: " + nama);
        System.out.println("Alamat: " + alamat);
        System.out.println();
        
        // 3. Konstanta (final)
        System.out.println ("3. KONSTANTA (FINAL)");
        System.out.println("---------------------");
        
        final double PI = 3.14159;
        final String NEGARA = "Indonesia";
        
        System.out.println ("Nilai PI: " + PI);
        System.out.println ("Negara: " + NEGARA);
        System.out.println();
        
        //4. Operasi dengan Variabel
        System.out.println("4. OPERASI DENGAN VARIABEL");
        System.out.println("--------------------------");
        
        int a = 10;
        int b = 5;
        
        int penjumlahan = a + b;
        int pengurangan = a - b;
        int perkalian = a * b;
        int pembagian = a / b;
        int modulus = a % b;
        
        System.out.println(a + " + " + b + " = " + penjumlahan);
        System.out.println(a + " - " + b + " = " + pengurangan);
        System.out.println(a + " * " + b + " = " + perkalian);
        System.out.println(a + " / " + b + " = " + pembagian);
        System.out.println(a + " % " + b + " = " + modulus);
        System.out.println();
        
        //5. Type Casting (konversi tipe data)
        System.out.println("5. TIPE CASTING");
        System.out.println("---------------");
        
        int angkaInt = 100;
        double angkaDoubel = angkaInt; // Widening casting (otomatis)
        int angkaKembali = (int) angkaDoubel; // Narrowing casting (manual)
        
        System.out.println("int ke double: " + angkaInt + " - " + angkaDoubel);
        System.out.println("double ke int: " + angkaDoubel + " - " + angkaKembali);
    }    
}
        

```
