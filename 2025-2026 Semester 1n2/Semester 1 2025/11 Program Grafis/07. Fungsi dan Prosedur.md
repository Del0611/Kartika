# 📚 Materi: Pengenalan Fungsi dan Prosedur (Java)

Saat membangun program yang besar, menulis semua kode di dalam satu blok `main` akan membuat kode menjadi panjang, rumit, dan sulit diperbaiki jika terjadi *error*. Untuk mengatasi masalah ini, kita menggunakan konsep **Fungsi** (*Function*) dan **Prosedur** (*Procedure*). 

Di dalam bahasa Java, kedua konsep ini dibungkus dalam satu istilah yang sama, yaitu **Method** (Metode). 🛠️

---

## 📦 Apa itu Method?

> 💡 **Definisi:** Method adalah blok kode terpisah yang dirancang untuk melakukan tugas spesifik tertentu. Kode di dalam method hanya akan dieksekusi ketika method tersebut dipanggil (*called*).

Dengan menggunakan method, kita bisa menerapkan prinsip **DRY (Don't Repeat Yourself)**. Cukup tulis kode sekali, dan Anda bisa menggunakannya berkali-kali di bagian program mana pun.

---

## 🔍 Perbedaan Fungsi dan Prosedur

Meskipun sama-sama disebut method di Java, secara konsep pemrograman keduanya memiliki perbedaan mendasar berdasarkan nilai kembaliannya:

### 1. Prosedur (Method Tanpa Nilai Balik)
Prosedur adalah method yang hanya melakukan serangkaian aksi/tugas tanpa mengembalikan nilai apa pun ke bagian program yang memanggilnya. Di Java, prosedur ditandai dengan kata kunci **`void`**.

* **Ciri khas:** Menggunakan kata kunci `void` dan **tidak** memiliki pernyataan `return`.

### 2. Fungsi (Method dengan Nilai Balik)
Fungsi adalah method yang melakukan perhitungan atau proses tertentu, lalu **mengembalikan hasil** (nilai) proses tersebut ke bagian program yang memanggilnya.

* **Ciri khas:** Tipe `void` diganti dengan **Tipe Data** dari nilai yang akan dikembalikan (seperti `int`, `double`, `String`, dll) dan **wajib** diakhiri dengan kata kunci **`return`**.

---

## 🏗️ Sintaks dan Struktur Dasar Method

```markdown
[modifier] [tipe_kembalian] namaMethod([parameter]) {
    // Blok kode / isi method
}

```

* **`modifier`**: Menentukan hak akses method (contoh: `public static`). Kata kunci `static` digunakan agar method bisa langsung dipanggil di dalam method `main` tanpa perlu membuat objek baru.
* **`tipe_kembalian`**: Diisi `void` jika berbentuk prosedur, atau diisi *tipe data* jika berbentuk fungsi.
* **`namaMethod`**: Nama unik untuk memanggil method tersebut (disarankan menggunakan gaya *camelCase* seperti `hitungLuas`).
* **`parameter` / Argumen**: Variabel opsional yang bertindak sebagai input data untuk diolah di dalam method.

---

## 💻 Implementasi Kode Java

Berikut adalah contoh program yang memisahkan struktur Prosedur (tanpa nilai balik) dan Fungsi (dengan nilai balik):

```java
public class BelajarMethod {

    // ==========================================
    // 1. CONTOH PROSEDUR (Menggunakan 'void')
    // ==========================================
    public static void sapaSiswa(String nama) {
        // Prosedur ini hanya bertugas mencetak teks ke layar
        System.out.println("👋 Halo " + nama + ", selamat pagi!");
    }

    // ==========================================
    // 2. CONTOH FUNGSI (Menggunakan Tipe Data)
    // ==========================================
    public static int hitungLuasPersegi(int sisi) {
        int luas = sisi * sisi;
        return luas; // Wajib mengembalikan nilai yang bertipe int
    }

    // ==========================================
    // UTAMA: Pintu Gerbang Program Dijalankan
    // ==========================================
    public static void main(String[] args) {
        
        System.out.println("=== 🚀 Memanggil Prosedur ===");
        // Memanggil prosedur 'sapaSiswa' dengan mengirimkan input string
        sapaSiswa("Andi");
        sapaSiswa("Budi");

        System.out.println("\n=== 📐 Memanggil Fungsi ===");
        // Memanggil fungsi 'hitungLuasPersegi' dan menyimpan hasilnya ke dalam variabel
        int hasilLuas = hitungLuasPersegi(5);
        System.out.println("Luas persegi dengan sisi 5 adalah: " + hasilLuas);
        
        // Fungsi juga bisa langsung dicetak tanpa disimpan ke variabel terlebih dahulu
        System.out.println("Luas persegi dengan sisi 10 adalah: " + hitungLuasPersegi(10));
    }
}

```

---

## 📌 Rangkuman untuk Diingat

| Karakteristik | Prosedur 🟢 | Fungsi 🔵 |
| --- | --- | --- |
| **Kata Kunci Tipe** | Wajib menggunakan `void` | Menggunakan tipe data seperti `int`, `String`, dll |
| **Pernyataan `return**` | Tidak memerlukan `return` | Wajib menggunakan `return` di akhir blok |
| **Tujuan Utama** | Melakukan aksi langsung (misal: cetak teks) | Melakukan kalkulasi data dan mengembalikan hasil |
| **Cara Pemanggilan** | Ditulis sebagai perintah mandiri | Biasanya dimasukkan ke variabel atau fungsi cetak |

```
