

# 📘 Variabel, Tipe Data, dan Operator (Python)

## 🧩 1. Variabel

Variabel adalah tempat untuk menyimpan data di dalam program. Data yang disimpan dalam variabel dapat digunakan kembali sesuai kebutuhan.

Dalam bahasa Python, variabel tidak perlu dideklarasikan secara khusus. Cukup dengan menuliskan nama variabel dan memberikan nilai.

**Contoh:**

```python
nama = "Budi"
umur = 15
tinggi = 160.5
```

Pada contoh di atas:

* `nama` menyimpan data teks
* `umur` menyimpan angka
* `tinggi` menyimpan angka desimal

---

## 🧩 2. Tipe Data

Tipe data adalah jenis nilai yang disimpan dalam variabel. Python memiliki beberapa tipe data dasar sebagai berikut:

### 🔹 a. Integer (int)

Digunakan untuk menyimpan bilangan bulat.

**Contoh:**

```python
jumlah = 10
```

---

### 🔹 b. Float (float)

Digunakan untuk menyimpan bilangan desimal.

**Contoh:**

```python
nilai = 85.5
```

---

### 🔹 c. String (str)

Digunakan untuk menyimpan teks.

**Contoh:**

```python
pesan = "Belajar Python itu mudah"
```

---

### 🔹 d. Boolean (bool)

Digunakan untuk menyimpan nilai benar atau salah.

**Contoh:**

```python
status = True
```

Nilai Boolean hanya terdiri dari:

* `True` (benar)
* `False` (salah)

---

## 🧩 3. Operator

Operator adalah simbol yang digunakan untuk melakukan operasi pada data.

### 🔹 a. Operator Aritmatika

Digunakan untuk perhitungan matematika.

| Operator | Fungsi      | Contoh |
| -------- | ----------- | ------ |
| `+`      | Penjumlahan | 2 + 3  |
| `-`      | Pengurangan | 5 - 2  |
| `*`      | Perkalian   | 4 * 3  |
| `/`      | Pembagian   | 10 / 2 |
| `%`      | Sisa bagi   | 5 % 2  |
| `**`     | Pangkat     | 2 ** 3 |

---

### 🔹 b. Operator Perbandingan

Digunakan untuk membandingkan dua nilai dan menghasilkan nilai benar atau salah.

| Operator | Arti                         |
| -------- | ---------------------------- |
| `==`     | Sama dengan                  |
| `!=`     | Tidak sama dengan            |
| `>`      | Lebih besar                  |
| `<`      | Lebih kecil                  |
| `>=`     | Lebih besar atau sama dengan |
| `<=`     | Lebih kecil atau sama dengan |

**Contoh:**

```python
print(10 > 5)   # True
print(10 == 7)  # False
```

---

### 🔹 c. Operator Logika

Digunakan untuk menggabungkan beberapa kondisi.

| Operator | Fungsi                                       |
| -------- | -------------------------------------------- |
| `and`    | Bernilai benar jika semua kondisi benar      |
| `or`     | Bernilai benar jika salah satu kondisi benar |
| `not`    | Membalik nilai                               |

**Contoh:**

```python
x = 10
print(x > 5 and x < 20)  # True
```

---

## 🎯 Kesimpulan

* Variabel digunakan untuk menyimpan data.
* Tipe data menentukan jenis nilai yang disimpan.
* Operator digunakan untuk melakukan operasi seperti perhitungan dan perbandingan.
* Pemahaman ketiga konsep ini sangat penting sebagai dasar dalam pemrograman Python.
