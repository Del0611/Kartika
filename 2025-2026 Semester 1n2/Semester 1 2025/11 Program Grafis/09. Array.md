# 📘 Menerapkan Array dan Struktur Data Sederhana (Python)

## 🧩 1. Pengertian Array (List)

Dalam Python, istilah array umumnya direpresentasikan menggunakan **list**. List adalah struktur data yang digunakan untuk menyimpan banyak nilai dalam satu variabel.

### 🔹 Ciri-ciri List:

* Dapat menyimpan lebih dari satu nilai
* Data dapat berbeda tipe
* Ditulis menggunakan tanda kurung siku `[]`
* Indeks dimulai dari 0

**Contoh:**

```python id="0j5l1g"
angka = [10, 20, 30, 40]
nama = ["Andi", "Budi", "Citra"]
campuran = [1, "Python", True]
```

---

## 🧩 2. Mengakses dan Mengubah Data List

### 🔹 a. Mengakses Data

Data dalam list diakses menggunakan indeks.

**Contoh:**

```python id="oaf2hc"
buah = ["Apel", "Jeruk", "Mangga"]
print(buah[0])  # Apel
```

---

### 🔹 b. Mengubah Data

Nilai dalam list dapat diubah.

**Contoh:**

```python id="g8q5bz"
buah = ["Apel", "Jeruk", "Mangga"]
buah[1] = "Pisang"
print(buah)
```

---

## 🧩 3. Operasi Dasar pada List

### 🔹 a. Menambah Data

```python id="4lxayl"
buah = ["Apel", "Jeruk"]
buah.append("Mangga")
```

---

### 🔹 b. Menghapus Data

```python id="9d4w8s"
buah = ["Apel", "Jeruk", "Mangga"]
buah.remove("Jeruk")
```

---

### 🔹 c. Mengetahui Panjang List

```python id="9t4k8p"
buah = ["Apel", "Jeruk", "Mangga"]
print(len(buah))
```

---

## 🧩 4. Perulangan pada List

List dapat digunakan bersama perulangan untuk mengakses seluruh data.

**Contoh:**

```python id="gqk2z1"
buah = ["Apel", "Jeruk", "Mangga"]

for item in buah:
    print(item)
```

---

## 🧩 5. Struktur Data Sederhana Lainnya

Selain list, Python juga memiliki struktur data sederhana lainnya:

---

### 🔹 a. Tuple

Tuple mirip dengan list, tetapi tidak dapat diubah (immutable).

**Contoh:**

```python id="b2jv1p"
data = (10, 20, 30)
```

---

### 🔹 b. Dictionary

Dictionary digunakan untuk menyimpan data dalam bentuk pasangan **kunci (key)** dan **nilai (value)**.

**Contoh:**

```python id="9nxq3f"
siswa = {
    "nama": "Andi",
    "umur": 16,
    "kelas": "XI"
}

print(siswa["nama"])
```

---

## 🧩 6. Praktik Sederhana

### 🔹 Studi Kasus: Menyimpan dan Menampilkan Data Nilai Siswa

```python id="k0r7jp"
nilai = [80, 75, 90, 85]

total = 0
for n in nilai:
    total += n

rata_rata = total / len(nilai)

print("Total nilai:", total)
print("Rata-rata:", rata_rata)
```

---

## 🎯 Kesimpulan

* Array dalam Python menggunakan **list**.
* List dapat menyimpan banyak data dalam satu variabel.
* Data dalam list dapat diakses, diubah, ditambah, dan dihapus.
* Struktur data lain seperti **tuple** dan **dictionary** memiliki fungsi yang berbeda.
* Penggunaan struktur data memudahkan pengolahan data dalam program.
