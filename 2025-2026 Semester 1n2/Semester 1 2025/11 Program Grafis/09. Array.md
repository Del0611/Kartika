# 📚 Materi: Pengenalan Array (1D & 2D) dan Praktik Pengolahan Data

Saat mengelola data yang banyak (misalnya daftar nilai siswa atau matriks koordinat), membuat variabel satu per satu seperti `nilai1`, `nilai2`, `nilai3` tentu sangat tidak efisien. Di sinilah kita membutuhkan **Array**. 🗂️

## 📦 BAGIAN 1: Pengenalan Array 1 Dimensi & 2 Dimensi

> 💡 **Definisi:** Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data dengan tipe data yang sama di dalam satu variabel. Setiap elemen di dalam array dapat diakses menggunakan nomor penunjuk yang disebut **Indeks**.

⚠️ **Aturan Emas Array:** Indeks array di dalam Java (dan hampir semua bahasa pemrograman) selalu dimulai dari angka **0**, bukan 1.

---

### 1. Array 1 Dimensi (1D) 📏
Array 1 Dimensi bisa dianalogikan seperti sebuah barisan kotak loker penyimpanan yang memanjang horizontal. Setiap loker memiliki nomor kunci (indeks).



#### A. Deklarasi dan Pembuatan Array 1D
Ada dua cara umum untuk membuat array 1D di Java:

* **Cara 1: Memesan ukuran array terlebih dahulu (Nilai diisi nanti)**
  ```java
  int[] nilaiSiswa = new int[5]; // Membuat array int berkapasitas 5 elemen
  


* **Cara 2: Langsung mengisi nilai elemen di awal**
```java
String[] namaBuah = {"Apel", "Mangga", "Pisang"};

```

#### B. Cara Mengakses dan Mengubah Elemen Array 1D

```java
String[] namaBuah = {"Apel", "Mangga", "Pisang"};

// Mengakses nilai (Output: Mangga)
System.out.println(namaBuah[1]); 

// Mengubah nilai elemen indeks ke-0 (Apel berubah menjadi Durian)
namaBuah[0] = "Durian";

```

---

### 2. Array 2 Dimensi (2D) 📊

Array 2 Dimensi sering disebut sebagai array di dalam array. Analoginya menyerupai bentuk **Tabel**, **Matriks**, atau papan catur yang memiliki baris dan kolom.

#### A. Deklarasi dan Pembuatan Array 2D

Sama seperti 1D, kita menggunakan tanda kurung siku ganda `[][]` untuk mewakili koordinat `[baris][kolom]`.

* **Cara 1: Memesan kapasitas baris dan kolom**
```java
int[][] matriks = new int[3][3]; // Membuat tabel berukuran 3 baris x 3 kolom

```


* **Cara 2: Langsung mengisi data tabel**
```java
int[][] koordinat = {
    {1, 2}, // Baris indeks 0
    {3, 4}, // Baris indeks 1
    {5, 6}  // Baris indeks 2
};

```



#### B. Cara Mengakses Elemen Array 2D

```java
// Mengakses angka 4 dari array 'koordinat' di atas (Baris indeks 1, Kolom indeks 1)
System.out.println(koordinat[1][1]); 

```

## 🛠️ BAGIAN 2: Praktik Pengolahan Data Menggunakan Array

Mari kita terapkan teori di atas ke dalam program nyata untuk mengolah data angka menggunakan teknik perulangan (*looping*).

### 1. Praktik Array 1D: Mencari Nilai Rata-Rata & Nilai Tertinggi

Program ini akan menghitung total nilai, mencari rata-rata, dan menyaring nilai tertinggi dari daftar nilai ujian siswa.

```java
public class OlahDataArray1D {
    public static void main(String[] args) {
        // Data nilai ujian 5 siswa
        int[] daftarNilai = {80, 95, 75, 90, 85};
        
        int total = 0;
        int nilaiTertinggi = daftarNilai[0]; // Asumsi awal, indeks ke-0 adalah yang tertinggi

        // Memproses array menggunakan perulangan for
        for (int i = 0; i < daftarNilai.length; i++) {
            total += daftarNilai[i]; // Menjumlahkan semua nilai
            
            // Logika mencari nilai tertinggi
            if (daftarNilai[i] > nilaiTertinggi) {
                nilaiTertinggi = daftarNilai[i];
            }
        }

        // Menghitung rata-rata (Gunakan double agar hasil desimal akurat)
        double rataRata = (double) total / daftarNilai.length;

        // Menampilkan Hasil Pengolahan
        System.out.println("=== REPORT DATA NILAI ===");
        System.out.println("📊 Total Nilai     : " + total);
        System.out.println("📈 Nilai Rata-rata : " + rataRata);
        System.out.println("🏆 Nilai Tertinggi : " + nilaiTertinggi);
    }
}

```

### 2. Praktik Array 2D: Mengolah Data Nilai Per Semester

Program ini memodelkan data nilai 3 siswa di mana masing-masing siswa memiliki 2 nilai mata pelajaran (Matematika dan Bahasa Inggris). Kita akan menampilkan datanya dalam bentuk tabel menggunakan **Nested Loop** (Perulangan Bersarang).

```java
public class OlahDataArray2D {
    public static void main(String[] args) {
        // Baris = Siswa (Budi, Andi, Cici)
        // Kolom = Nilai Mapel (Kolom 0: Matematika, Kolom 1: B. Inggris)
        int[][] nilaiSiswa = {
            {80, 85}, // Nilai Budi
            {90, 75}, // Nilai Andi
            {95, 90}  // Nilai Cici
        };

        String[] namaSiswa = {"Budi", "Andi", "Cici"};

        System.out.println("=========================================");
        System.out.println("NAMA\t\tMATEMATIKA\tB. INGGRIS");
        System.out.println("=========================================");

        // Outer loop untuk mengontrol baris (Siswa)
        for (int baris = 0; baris < nilaiSiswa.length; baris++) {
            
            // Cetak nama siswa terlebih dahulu
            System.out.print(namaSiswa[baris] + "\t\t");
            
            // Inner loop untuk mengontrol kolom (Nilai Mapel)
            for (int kolom = 0; kolom < nilaiSiswa[baris].length; kolom++) {
                System.out.print(nilaiSiswa[baris][kolom] + "\t\t");
            }
            
            System.out.println(); // Pindah ke baris baru di konsol setelah kolom habis
        }
        System.out.println("=========================================");
    }
}

```

## 📌 Rangkuman Penting

* Ukuran array di Java bersifat **statis** (tetap), jika sudah ditentukan kapasitasnya sebesar 5, maka tidak bisa ditambah menjadi 6 di tengah jalan.
* Gunakan properti `.length` untuk mengetahui jumlah panjang atau kapasitas dari suatu array secara otomatis (contoh: `namaArray.length`).
* Untuk memproses seluruh isi tabel Array 2D secara merata, pola perulangan bersarang (*nested loop for*) adalah solusi yang paling standar digunakan.
