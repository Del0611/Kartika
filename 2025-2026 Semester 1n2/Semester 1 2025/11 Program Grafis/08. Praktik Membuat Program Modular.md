# 📚 Materi: Praktik Membuat Program Modular (Java)

Setelah memahami konsep fungsi dan prosedur (method), sekarang saatnya kita menerapkannya ke dalam bentuk **Pemrograman Modular**.

---

## 🧩 Apa itu Pemrograman Modular?

> 💡 **Definisi:** Pemrograman modular adalah teknik pembuatan program dengan cara memecah program besar menjadi bagian-bagian kecil (modul/method) yang lebih fokus pada satu tugas spesifik.

### 🌟 Keuntungan Program Modular:
1. **Reusability:** Modul yang sudah dibuat bisa digunakan berulang kali tanpa menulis ulang kodenya.
2. **Mudah Didebug:** Jika terjadi kesalahan (*bug*), kita tahu persis modul mana yang bermasalah.
3. **Kode Lebih Rapi:** Fungsi utama (`main`) tidak penuh sesak oleh ratusan baris kode.

---

## 🛠️ Studi Kasus: Aplikasi Kalkulator Geometri

Kita akan mempraktikkan gaya modular ini untuk membuat program kalkulator matematika yang dapat menghitung **Luas Persegi**, **Luas Lingkaran**, dan **Luas Segitiga**.

### 📐 Alur Logika Program (Modular)
Kita tidak akan menumpuk semua rumus di dalam method `main`. Kita akan membaginya ke dalam 4 modul terpisah:
* 🟢 **`hitungLuasPersegi`** (Fungsi dengan nilai balik)
* 🔵 **`hitungLuasLingkaran`** (Fungsi dengan nilai balik)
* 🟡 **`hitungLuasSegitiga`** (Fungsi dengan nilai balik)
* 📱 **`tampilkanMenu`** (Prosedur tanpa nilai balik untuk antarmuka pengguna)

---

## 💻 Implementasi Kode Sumber Java

Salin seluruh kode di bawah ini ke dalam file bernama `KalkulatorGeometri.java`. Program ini juga memanfaatkan *class* `Scanner` untuk menerima input dinamis dari terminal/konsol.

```java
import java.util.Scanner;

public class KalkulatorGeometri {

    // ==========================================
    // 1. MODUL / METHOD PROSEDUR (Tampilkan Menu)
    // ==========================================
    public static void tampilkanMenu() {
        System.out.println("\n=================================");
        System.out.println("   ⚙️ APLIKASI GEOMETRI MODULAR   ");
        System.out.println("=================================");
        System.out.println("1. Hitung Luas Persegi ⏹️");
        System.out.println("2. Hitung Luas Lingkaran ⏸️");
        System.out.println("3. Hitung Luas Segitiga 📐");
        System.out.println("4. Keluar Aplikasi ❌");
        System.out.print("Pilih menu (1-4): ");
    }

    // ==========================================
    // 2. MODUL / METHOD FUNGSI (Hitung Luas Persegi)
    // ==========================================
    public static double hitungLuasPersegi(double sisi) {
        return sisi * sisi;
    }

    // ==========================================
    // 3. MODUL / METHOD FUNGSI (Hitung Luas Lingkaran)
    // ==========================================
    public static double hitungLuasLingkaran(double jariJari) {
        // Menggunakan konstanta bawaan Java Math.PI (3.14159...)
        return Math.PI * jariJari * jariJari;
    }

    // ==========================================
    // 4. MODUL / METHOD FUNGSI (Hitung Luas Segitiga)
    // ==========================================
    public static double hitungLuasSegitiga(double alas, double tinggi) {
        return 0.5 * alas * tinggi;
    }

    // ==========================================
    // METHOD UTAMA (Pusat Kendali Program)
    // ==========================================
    public static void main(String[] args) {
        Scanner input = new Scanner(System.util.in);
        int pilihan;
        boolean berjalan = true;

        while (berjalan) {
            // Memanggil modul prosedur untuk memunculkan menu teks
            tampilkanMenu();
            pilihan = input.nextInt();

            switch (pilihan) {
                case 1:
                    System.out.print("Masukkan panjang sisi persegi: ");
                    double s = input.nextDouble();
                    // Memanggil fungsi luas persegi
                    double hasilPersegi = hitungLuasPersegi(s);
                    System.out.printf("👉 Luas Persegi: %.2f\n", hasilPersegi);
                    break;

                case 2:
                    System.out.print("Masukkan panjang jari-jari lingkaran: ");
                    double r = input.nextDouble();
                    // Memanggil fungsi luas lingkaran
                    double hasilLingkaran = hitungLuasLingkaran(r);
                    System.out.printf("👉 Luas Lingkaran: %.2f\n", hasilLingkaran);
                    break;

                case 3:
                    System.out.print("Masukkan alas segitiga: ");
                    double a = input.nextDouble();
                    System.out.print("Masukkan tinggi segitiga: ");
                    double t = input.nextDouble();
                    // Memanggil fungsi luas segitiga
                    double hasilSegitiga = hitungLuasSegitiga(a, t);
                    System.out.printf("👉 Luas Segitiga: %.2f\n", hasilSegitiga);
                    break;

                case 4:
                    System.out.println("Terima kasih! Program selesai. 👋");
                    berjalan = false;
                    break;

                default:
                    System.out.println("⚠️ Pilihan tidak valid. Silakan coba lagi.");
            }
        }
        input.close(); // Menutup resource scanner
    }
}

```

---

## 🔍 Analisis Pola Kerja Kode

Coba perhatikan blok `switch-case` di dalam fungsi `main`. Kode di sana terlihat sangat bersih dan mudah dibaca:

```java
case 1:
    System.out.print("Masukkan panjang sisi persegi: ");
    double s = input.nextDouble();
    double hasilPersegi = hitungLuasPersegi(s); // <--- Tinggal panggil modulnya saja
    System.out.printf("👉 Luas Persegi: %.2f\n", hasilPersegi);
    break;

```

Jika suatu hari rumus luas lingkaran perlu diubah atau disesuaikan (misalnya pembatasan nilai desimal atau penambahan validasi angka minus), Anda **tidak perlu mengutak-atik fungsi `main**`. Anda hanya perlu melompat langsung ke blok tubuh method `hitungLuasLingkaran`.

---

## 📌 Kesimpulan Praktik

Melalui pemrograman modular, kode kita menjadi bersifat terisolasi dan mandiri (*decoupled*). Struktur ini mempermudah kolaborasi tim saat membangun sistem perangkat lunak berskala besar, di mana setiap programmer dapat fokus mengerjakan modulnya masing-masing.
