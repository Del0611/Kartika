
# 📘 Pengenalan Fungsi dan Prosedur serta Program Modular (Python)

## 🧩 1. Pengertian Fungsi dan Prosedur

Dalam pemrograman, fungsi dan prosedur digunakan untuk mengelompokkan perintah agar program lebih terstruktur dan mudah dikelola.

### 🔹 a. Fungsi

Fungsi adalah blok kode yang digunakan untuk melakukan suatu tugas tertentu dan dapat mengembalikan nilai (return).

**Ciri-ciri fungsi:**

* Memiliki nama
* Dapat menerima input (parameter)
* Menghasilkan keluaran (return)

**Contoh:**

```python
def hitung_luas_persegi(sisi):
    luas = sisi * sisi
    return luas

hasil = hitung_luas_persegi(5)
print("Luas:", hasil)
```

---

### 🔹 b. Prosedur

Prosedur adalah blok kode yang digunakan untuk melakukan suatu tugas, tetapi tidak mengembalikan nilai.

**Contoh:**

```python
def sapa(nama):
    print("Halo,", nama)

sapa("Andi")
```

---

## 🧩 2. Manfaat Fungsi dan Prosedur

Penggunaan fungsi dan prosedur memiliki beberapa manfaat, yaitu:

* Program menjadi lebih rapi dan terstruktur
* Mengurangi pengulangan kode
* Memudahkan pemeliharaan program
* Mempermudah kerja sama dalam tim

---

## 🧩 3. Program Modular

Program modular adalah program yang disusun menjadi beberapa bagian kecil (modul), seperti fungsi atau prosedur, sehingga lebih mudah dikembangkan dan dipahami.

---

## 🧩 4. Praktik Membuat Program Modular

Berikut contoh program sederhana yang menggunakan konsep modular:

### 🔹 Studi Kasus: Menghitung Luas dan Keliling Persegi

```python
# fungsi menghitung luas
def hitung_luas(sisi):
    return sisi * sisi

# fungsi menghitung keliling
def hitung_keliling(sisi):
    return 4 * sisi

# program utama
sisi = int(input("Masukkan panjang sisi: "))

luas = hitung_luas(sisi)
keliling = hitung_keliling(sisi)

print("Luas persegi:", luas)
print("Keliling persegi:", keliling)
```

---

### 🔹 Penjelasan:

* Program dibagi menjadi beberapa fungsi:

  * `hitung_luas()` untuk menghitung luas
  * `hitung_keliling()` untuk menghitung keliling
* Bagian utama program hanya memanggil fungsi yang diperlukan

---

## 🎯 Kesimpulan

* Fungsi adalah blok kode yang dapat mengembalikan nilai.
* Prosedur adalah blok kode yang tidak mengembalikan nilai.
* Program modular adalah program yang dibagi menjadi bagian-bagian kecil.
* Penggunaan fungsi dan prosedur membuat program lebih rapi, efisien, dan mudah dipahami.
