## 📥 BAGIAN 1: Input/Output Teks

Di dalam Java, operasi standar Input dan Output (I/O) berbasis teks dilakukan melalui *stream* (aliran data) bawaan yang dikelola oleh `System class`.

### 1. Menampilkan Output Teks
Java menyediakan dua perintah utama yang paling sering digunakan untuk menampilkan teks ke konsol/terminal:

*   **`System.out.print()`**: Menampilkan teks ke layar tanpa membuat baris baru. Teks berikutnya akan muncul tepat di samping teks sebelumnya.
*   **`System.out.println()`**: Menampilkan teks ke layar sekaligus membuat baris baru (*line break*). Teks berikutnya akan dicetak di baris bawahnya.

#### 💡 Perbandingan Print vs Println:
```java
System.out.print("Satu");
System.out.print("Dua");
// Output: SatuDua

System.out.println("Tiga");
System.out.println("Empat");
// Output:
// Tiga
// Empat

```

### 2. Menerima Input Teks dengan `Scanner`

Untuk membaca apa yang diketik oleh pengguna melalui keyboard, Java menyediakan sebuah *library* bawaan bernama **`Scanner`** yang berada di dalam paket `java.util`.

#### 🏗️ Alur Menggunakan Scanner:

1. Impor library `Scanner` di baris paling atas file kode.
2. Buat objek `Scanner` baru di dalam method `main`.
3. Gunakan method yang sesuai dengan tipe data yang ingin dibaca (misal: `.nextLine()` untuk teks, `.nextInt()` untuk angka bulat).

#### 💻 Contoh Kode Praktik Input/Output Teks:

```java
// Langkah 1: Import library Scanner
import java.util.Scanner;

public class PraktikIO {
    public static void main(String[] args) {
        // Langkah 2: Membuat objek Scanner bernama 'input'
        Scanner input = new Scanner(System.in);
        
        System.out.println("=== 📝 PROGRAM REGISTRASI SISWA ===");
        
        // Langkah 3: Mengambil input bertipe String (Teks)
        System.out.print("Masukkan Nama Lengkap: ");
        String nama = input.nextLine();
        
        // Langkah 3: Mengambil input bertipe Integer (Angka Bulat)
        System.out.print("Masukkan Umur: ");
        int umur = input.nextInt();
        
        System.out.println("\n-----------------------------------");
        // Menampilkan Output hasil input data
        System.out.println("🎉 Registrasi Berhasil!");
        System.out.println("Halo " + nama + ", umur Anda sekarang adalah " + umur + " tahun.");
        
        // Selalu tutup scanner di akhir program untuk menghemat memori
        input.close();
    }
}

```

## 🎨 BAGIAN 2: Pengenalan Library Grafik Sederhana

Selain berbasis teks hitam-putih di terminal, Java juga memiliki kapabilitas untuk membuat Antarmuka Pengguna Grafis (GUI) secara visual. Library bawaan klasik yang sangat cocok untuk mempelajari konsep dasar grafik dan koordinat adalah **AWT (Abstract Window Toolkit)** dan **Swing**.

### 1. Konsep Koordinat Layar Grafik Java

Sebelum menggambar objek visual seperti kotak atau lingkaran, Anda harus memahami bahwa komputer menggunakan sistem koordinat kartesius yang sedikit berbeda dengan matematika biasa:

* Titik pusat `(0,0)` berada di **pojok kiri atas** jendela aplikasi.
* Sumbu **X** akan semakin besar jika bergeser ke arah **kanan**.
* Sumbu **Y** akan semakin besar jika bergeser ke arah **bawah**.

---

### 2. Praktik Menggambar Bentuk Dua Dimensi (2D)

Kita akan membuat sebuah jendela aplikasi visual (*Frame*) dan menggambar beberapa bentuk dasar seperti Garis, Persegi, dan Lingkaran menggunakan komponen `JFrame` dan `Graphics`.

#### 💻 Contoh Kode Praktik Jendela Grafik:

```java
import javax.swing.JFrame;
import java.awt.Graphics;
import java.awt.Color;

// Membuat class yang mewarisi (extends) sifat dari JFrame
public class JendelaGrafikSederhana extends JFrame {

    // Constructor untuk mengatur konfigurasi jendela aplikasi
    public JendelaGrafikSederhana() {
        setTitle("Aplikasi Jendela Grafik Sederhana 🎨");
        setSize(400, 400); // Lebar 400 piksel, Tinggi 400 piksel
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); // Program berhenti total saat jendela ditutup
        setLocationRelativeTo(null); // Membuat jendela muncul di tengah layar monitor
    }

    // Method khusus bawaan Java untuk menggambar elemen visual
    @Override
    public void paint(Graphics g) {
        super.paint(g); // Memastikan fungsi dasar menggambar dari sistem tetap berjalan
        
        // 1. Menggambar GARIS LURUS (Warna Merah)
        g.setColor(Color.RED);
        // Parameter: (x1, y1, x2, y2)
        g.drawLine(50, 80, 350, 80);
        
        // 2. Menggambar KOTAK / PERSEGI PANJANG (Warna Biru)
        g.setColor(Color.BLUE);
        // Parameter: (x, y, lebar, tinggi)
        g.drawRect(50, 120, 120, 80);
        
        // 3. Menggambar KOTAK BERWARNA PENUH (Warna Hijau)
        g.setColor(Color.GREEN);
        g.fillRect(220, 120, 120, 80);
        
        // 4. Menggambar LINGKARAN / OVAL BERWARNA PENUH (Warna Kuning)
        g.setColor(Color.ORANGE);
        // Parameter: (x, y, lebar_oval, tinggi_oval)
        g.fillOval(135, 240, 120, 120);
        
        // 5. Menampilkan TEKS di dalam Area Grafik
        g.setColor(Color.BLACK);
        g.drawString("Halo dari dunia GUI Java!", 130, 70);
    }

    // Method Utama untuk menjalankan program grafik
    public static void main(String[] args) {
        // Membuat objek jendela
        JendelaGrafikSederhana jendela = new JendelaGrafikSederhana();
        // Mengaktifkan visibilitas jendela agar muncul di layar monitor
        jendela.setVisible(true);
    }
}

```

## 📌 Rangkuman Penting

* Library `Scanner` membutuhkan penanganan tipe data yang tepat; jika pengguna memasukkan teks huruf saat program meminta `.nextInt()`, program akan mengalami *InputMismatchException* (error).
* Dalam pemrograman grafik Java, method `paint(Graphics g)` dipanggil secara otomatis oleh sistem operasi saat jendela pertama kali dirender atau ketika ukurannya diubah (*resize*).
* Perintah berawalan `draw` (seperti `drawRect`) hanya akan menggambar garis tepi/rangka objek saja, sedangkan perintah berawalan `fill` (seperti `fillRect`) akan mewarnai seluruh bagian dalam objek secara penuh.
