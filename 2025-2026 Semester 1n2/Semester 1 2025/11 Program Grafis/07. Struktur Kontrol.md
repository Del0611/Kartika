# 📘 Struktur Kontrol (If, Else, Loop) dalam Python

Struktur kontrol adalah bagian dari program yang digunakan untuk mengatur alur eksekusi perintah. Dengan struktur kontrol, program dapat mengambil keputusan dan melakukan perulangan.

---

## 🧩 1. Percabangan (If, Else)

Percabangan digunakan untuk menentukan tindakan berdasarkan suatu kondisi.

### 🔹 a. If

`if` digunakan untuk menjalankan perintah jika kondisi bernilai benar (`True`).

**Contoh:**

```python
nilai = 80

if nilai >= 75:
    print("Lulus")
```

---

### 🔹 b. If - Else

`if-else` digunakan untuk memilih dua kemungkinan kondisi.

**Contoh:**

```python
nilai = 60

if nilai >= 75:
    print("Lulus")
else:
    print("Tidak lulus")
```

---

### 🔹 c. If - Elif - Else

Digunakan jika terdapat lebih dari dua kondisi.

**Contoh:**

```python
nilai = 85

if nilai >= 90:
    print("A")
elif nilai >= 75:
    print("B")
elif nilai >= 60:
    print("C")
else:
    print("D")
```

---

## 🧩 2. Perulangan (Loop)

Perulangan digunakan untuk menjalankan perintah secara berulang.

---

### 🔹 a. For Loop

Digunakan untuk perulangan dengan jumlah yang sudah diketahui.

**Contoh:**

```python
for i in range(5):
    print("Perulangan ke-", i)
```

Penjelasan:

* `range(5)` menghasilkan angka dari 0 sampai 4
* Perulangan dilakukan sebanyak 5 kali

---

### 🔹 b. While Loop

Digunakan untuk perulangan selama kondisi bernilai benar.

**Contoh:**

```python
i = 0

while i < 5:
    print("Angka:", i)
    i += 1
```

Penjelasan:

* Perulangan akan berhenti ketika kondisi `i < 5` bernilai salah

---

## 🧩 3. Pernyataan Khusus dalam Loop

### 🔹 a. Break

Digunakan untuk menghentikan perulangan secara langsung.

**Contoh:**

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

---

### 🔹 b. Continue

Digunakan untuk melewati satu iterasi dan melanjutkan ke iterasi berikutnya.

**Contoh:**

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

---

## 🎯 Kesimpulan

* Struktur kontrol digunakan untuk mengatur alur program.
* Percabangan (`if`, `else`, `elif`) digunakan untuk pengambilan keputusan.
* Perulangan (`for`, `while`) digunakan untuk menjalankan perintah secara berulang.
* `break` dan `continue` membantu mengontrol jalannya perulangan.
